    ➤ Tools ◬¶

    Коды возмещения

    Выше линк на офф сайт для активации кодов, так же есть open source iOS|Android приложение Которое будет уведомлять о всех выпущенных кодах и активировать без необходимости посещать сайт Lilith. Так что вы никогда не пропустите ни одного кода!

    AFCalculator и не только

    Много полезный инструментов, создание оценочных листов. Калькуляторы:

    • для бездны выпадения/продажи реликвий
    • сравнения value ресурсов
    • ИП эмблем
    • АФК и Быстрых наград
    AFK Helper

    База проходок кампании/башек. Теряет актуальность по мере снижения сложности в кампании. Но все еще остается полезной для поиска адекватных проходок, особенно на ранних стадиях

    AbEx relic drop

    Еще один калькулятор дроп реликвий в бездне

    Engravings Calculator

    Калькулятор для примерного расчета прироста гравировок.

    Elder Tree Planner

    Калькулятор для примерного расчета прироста сока(дерево), показывает временные рамки до капа личного/общего.
